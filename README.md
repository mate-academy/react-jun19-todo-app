# React ToDo App

Todo App on React

### Functionality

* Create task
* Double click to edit task
* Complete task
* Delete task
* Complete all tasks

## Used technologies
* React
* CSS3

## DEMO
* [DEMO LINK](https://velzepooz.github.io/react_todo-app/)

## Run
* npm i - install
* npm run - starts project on localhost
* npm run build - production build
